How to use?
1. Create db table with the help of db_short_urls.sql
2. place .htacess and redirect in same path
3. to Shorten the URL
- got to <domain_name>/url_shortner/shrt_url.php?url=<url_to_short>
e.g 127.0.0.1//url_shortner/shrt_url.php?url=google.com
    - it will return 127.0.0.1/1
    - ..............
    - 127.0.0.1/fz
4. Type the 127.0.0.1/1 in url, it will redirect you to google.com as per our example
